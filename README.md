# Prissma Café Website

Este proyecto contiene el código fuente para el sitio web de **Prissma Café**.

## Estructura del Proyecto

- `index.html`: Página principal.
- `terminos.html`: Página de Términos y Condiciones.
- `privacidad.html`: Política de Privacidad.
- `css/styles.css`: Hoja de estilos principal.
- `js/script.js`: Lógica de la interfaz (menú móvil, animaciones).
- `assets/images/`: Imágenes generadas y recursos.
- `assets/videos/`: Carpeta para videos (actualmente usa un video externo de ejemplo en el código, puedes descargar uno real aquí).

## Cómo visualizar

Simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Edge, Firefox).

## Personalización

1. **Video de Fondo**: Reemplaza la URL en la etiqueta `<video>` en `index.html` por una ruta local `assets/videos/tu-video.mp4` para mejor rendimiento.
2. **Imágenes**: Las imágenes actuales fueron generadas por IA. Puedes reemplazarlas por fotos reales de tu local en la carpeta `assets/images`.

## Tecnologías

- HTML5 Semántico
- CSS3 (Variables, Flexbox, Grid, Animaciones)
- Vanilla JavaScript

Diseñado para ser ligero, rápido y fácil de mantener.
